# 💬 Site de Fofoca — Projeto Web (1º semestre)
📌 Sobre o projeto

Este é um site de fofoca desenvolvido como trabalho acadêmico no 1º semestre do curso de Desenvolvimento de Software Multiplataforma.

Foi o meu primeiro projeto web, criado com o objetivo de aplicar os conceitos iniciais de HTML e CSS, construindo uma interface completa, com navegação por abas, imagens, textos e organização de conteúdo.

O site simula um portal de entretenimento, reunindo notícias e curiosidades de forma leve, visual e interativa.

---

# Objetivos do trabalho
Praticar estruturação de páginas com HTML semântico
Aprender estilização com CSS
Criar layouts organizados e agradáveis
Desenvolver navegação por seções/abas
Implementar responsividade para diferentes telas
Compreender organização de arquivos web

---

## Tecnologias utilizadas:
- HTML
- CSS
- JS

---

# Funcionalidades

- Layout com múltiplas seções de conteúdo
- Sistema de abas para navegação
- Cards com imagens e legendas
- Menu de navegação superior
- Efeitos visuais (hover, sombras, destaques)
- Design responsivo para celular, tablet e desktop
- Botão “voltar ao topo”
